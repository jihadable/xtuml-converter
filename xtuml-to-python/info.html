This application helps you process JSON xtUML files into Python code by supporting various data types and models.

Main features

  • Converts xtUML JSON to Python classes.
  • Support list, tuple, enum data type.
  • Validate the JSON structure to ensure correct input.
  • Interactive interface for uploading files and previewing JSON.
  • The output results can be downloaded or copied easily.
